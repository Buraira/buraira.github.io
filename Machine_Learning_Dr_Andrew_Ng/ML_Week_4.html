Neural Networks

I am not going to describe here what a neuron is. But I will write some points about a neuron to get a better understand on neural network.

  1. A neuron has dendrites as input signals and axons as output signals.
  2. It has a main body for funtions.
  3. Neuron transmits electrical signals within each other and odes functions to perform tasks on the whole body using those three things.
In neural networks there are 3 layers.
  1. input layer
  2. hidden layer
  3. output layer

The hidden layer may be of multiple numbers. In the each layer there are bias units with value of 1. The hidden layer nodes are called activation unit ai(j). We have to calculate the activation function to go to the next layer.
